# **App Name**: FastTrackJob

## Core Features:

- AI-Powered Candidate Matching: Utilize a Zero-Storage AI tool to instantly evaluate candidates against job requirements without retaining sensitive data long-term, generating real-time match scores.
- Live Applicant Leaderboard: Display a dynamic leaderboard showing candidates ranked by their AI-generated match scores, offering quick visualization of top talent.
- Job Posting and Management: Allow HR users to post new job listings, manage existing ones, and set criteria for applicant pre-screening.
- Job Seeker Application Interface: Provide a clean interface for job seekers to find available positions and submit their applications, with immediate feedback on eligibility.
- Automated Application Filtering: Implement an auto-cutoff quota tool to automatically filter out applications that do not meet pre-defined minimum criteria, saving HR time.
- Automated Interview Scheduling: Facilitate the automatic scheduling of interviews for top-matched candidates, streamlining the recruitment workflow.

## Style Guidelines:

- Primary color (brand and key text): A deep, professional blue (#002147), reflecting reliability and seriousness. (Derived from HSL(217, 100%, 13.9%))
- Background color: A very light, desaturated blue (#F4F6FB) for a clean, modern aesthetic that ensures high readability. (Derived from HSL(217, 20%, 98%))
- Accent color (calls to action, highlights): A vibrant cyan-teal (#006a6a) to convey innovation, speed, and create visual anchors for user interaction. (Derived from HSL(180, 100%, 20%))
- Headline and Body font: 'Inter' (sans-serif) for its modern, clear, and highly readable characteristics across all content types. Note: currently only Google Fonts are supported.
- Utilize Google Material Symbols (Outlined variant) for their consistency, versatility, and clear communication of actions and statuses.
- Content should be centralized within a `max-w-[1240px]` container, utilizing a consistent `px-6` horizontal padding. Sections employ `flex` and `grid` layouts for responsive structuring across devices.
- Incorporate subtle animations like `float` for decorative elements, `gentle-pulse` for important status indicators, `fade-in-up` for content on load, and a `shimmer-effect` for loading states to enhance user engagement and indicate dynamic content.
